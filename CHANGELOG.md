# Changelog

## v1.1.0
- Release polish and documentation hardening.
- Added explicit project-status, compatibility, dialect, comparison, and package-map docs.
- Tightened the README positioning around safety, trust, and public API boundaries.
- Hardened the Makefile and CI checks, including example verification and race coverage.
- Expanded examples into longer product-style walkthroughs.
- Added and strengthened safety, dialect, scan, Codex, and public API tests.
- Clarified the optional `scan`, `hydra`, and `codex` package surfaces.

## v1.0.0
- Initial public release of Quarry.
- Fluent SELECT, INSERT, UPDATE, and DELETE builders.
- Dialect-aware placeholders and identifier quoting.
- Dynamic predicates and safe sorting helpers.
- Raw SQL fragments with bound arguments.
- Optional scanning helpers.
- Optional Codex query/recipe registry.
- Examples and integration docs.
